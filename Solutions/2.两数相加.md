# 两数相加
链表简单题，注意题目的理解，倒序存放，存储进位直接相加即可。
+ 创建哑节点，更加容易写
``` python
class Solution:
    def addTwoNumbers(self, l1: ListNode, l2: ListNode) -> ListNode:
        add = 0
        dummyNode = ListNode(0)
        dummyNode.next = None
        cur = dummyNode
        while l1 or l2:
            x = l1.val if l1 else 0
            y = l2.val if l2 else 0
            result = x + y + add
            add = result // 10
            temp = ListNode(result % 10)
            cur.next = temp
            cur = cur.next
            if l1:
                l1 = l1.next
            if l2:
                l2 = l2.next
        if add:
            temp = ListNode(add)
            cur.next = temp
            cur = cur.next
        cur.next = None

        return dummyNode.next
```
